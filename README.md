# Snack Sense

Snack Sense is a fun little web app where you crack open a virtual fortune cookie to reveal a random snack prediction.  
Every snack belongs to a category (sweet, sour, salty, fruity, spicy, chocolatey, crispy, savory, misc), and your ratings help shape what you see next.

The more you use it, the better it learns your taste.

---

## Features

- **Fortune Cookie Animation**  
  Click the cookie to crack it open and reveal your snack destiny.

- **300+ Snacks Built‑In**  
  All snacks have proper categories.
  
- **Smart Recommendation System**  
  - ❤️ Love = +2  
  - 😐 Meh = 0  
  - 💀 Hate = –2  
  Your ratings influence future predictions.

- **Category Weighting**  
  Love sour snacks? You’ll see more sour.  
  Hate spicy snacks? They’ll show up less.

- **Stats Modal**  
  Shows your total runs, category scores, and top‑rated snacks.

- **Custom Snacks**  
  Add your own snacks with categories.

- **Local Storage**  
  All data is saved in your browser — no backend required.

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla)  
- LocalStorage for persistence  

No frameworks, no dependencies — just clean front‑end code.
