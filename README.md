# Snack Sense

Snack Sense is a sort of useful web app that predicts what snack you're craving. 
It starts as a simple virtual fortune cookie, but over time it becomes personalised; users rate each prediction and they system learns their preferences to generate more accurate suggestion. 
Made with HTML, CSS and JS with all data saved locally on device :D

## Features

- Predicts a snack from a fortune-cookie style interaction.
- Learns from user feedback with Love, Meh, and HATE ratings.
- Keeps stats locally in the browser.
- Lets users add custom snacks to the prediction pool.
- Tracks achievements and shows unlock notifications.
- Supports light and dark themes.
- Exports a copy of your data, including exporting sharable images! 
- Lets you reset all saved data when you want a fresh start.

## How To Use

Open the page, click the fortune cookie, and rate the snack you get. The more you use it, the more the app learns your preferences.

## Tech Stack

Snack Sense is built with vanilla HTML, CSS, and JavaScript, with data stored locally in the browser.
